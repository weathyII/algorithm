我们可以注意到，如果 ⌊ab⌋ = a mod b = k，则 a 可以表示为 kb + k（b > k）。由于 b > k，我们有 k² < kb + k = a ≤ x。因此，k ≤ √x。

现在让我们统计对于任意固定的 k（1 ≤ k ≤ √x）下的特殊对数目。对于每个 k，你需要统计满足以下条件的 b 的个数：  
1. b > k  
2. 1 ≤ b ≤ y  
3. 1 ≤ kb + k ≤ x  

第三个条件等价于 1 ≤ b ≤ x/k − 1。

因此，对于任意固定的 k > 0，满足条件的特殊对 (a ≤ x; b ≤ y) 的个数为 max(0, min(y, x/k − 1) − k)。

最终结果是每个 k 的特殊对数量的总和。

复杂度：O(√x)。
